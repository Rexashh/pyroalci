
#  ♞ Pyronath-Userbot ♞

Pyronath-Userbot adalah userbot Telegram modular yang berjalan di Python3 dengan Library Pyrogram.

```

Ketika anda sudah memasang userbot ini, berarti anda sudah siap dengan resikonya.
```
